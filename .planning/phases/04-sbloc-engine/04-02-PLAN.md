  <action>
Create LTV (Loan-to-Value) calculation utilities:

**calculateLTV(loanBalance: number, collateralValue: number): number**
- Returns loan / collateral ratio
- Handle edge case: collateralValue <= 0 returns Infinity (or 1.0 if loan is also 0)
- Example: $500k loan / $1M collateral = 0.50 (50%)

**calculateMaxBorrowing(collateralValue: number, maxLTV: number): number**
- Returns maximum borrowable amount
- Formula: collateralValue * maxLTV
- Example: $1M collateral * 0.65 maxLTV = $650k max borrowing

**getEffectiveLTV(portfolio: { assets: Array<{ value: number; assetClass: 'equity' | 'bond' | 'cash' }> }, ltvLimits: LTVByAssetClass): number**
- Calculate weighted average LTV limit for a portfolio
- Each asset contributes its value * its class LTV limit to total borrowing power
- Return: totalBorrowingPower / totalPortfolioValue
- Example: $800k equities (50% LTV) + $200k bonds (90% LTV) = ($400k + $180k) / $1M = 58% effective LTV

**isWithinBorrowingLimit(state: SBLOCState, config: SBLOCConfig): boolean**
- Returns true if currentLTV < maxLTV
- Simple threshold check

**calculateAvailableCredit(state: SBLOCState, config: SBLOCConfig): number**
- Returns remaining borrowing capacity
- Formula: (portfolioValue * maxLTV) - loanBalance
- Floor at 0 (can't have negative available credit)

Include JSDoc with example calculations. Pure functions, no mutations.

Create margin call detection logic:

**detectMarginCall(state: SBLOCState, config: SBLOCConfig): MarginCallEvent | null**
- Returns MarginCallEvent if currentLTV >= maxLTV, null otherwise
- Event contains: year (from state), ltvAtTrigger, loanBalance, portfolioValue
- This is the critical threshold that triggers forced liquidation

**isInWarningZone(state: SBLOCState, config: SBLOCConfig): boolean**
- Returns true if: maintenanceMargin <= currentLTV < maxLTV
- The "yellow zone" before margin call
- Used for UI warning indicators

**calculateMarginBuffer(state: SBLOCState, config: SBLOCConfig): { dollarsUntilWarning: number; dollarsUntilMarginCall: number; percentUntilMarginCall: number }**
- How much portfolio can drop before warning/margin call
- dollarsUntilWarning = portfolioValue - (loanBalance / maintenanceMargin)
- dollarsUntilMarginCall = portfolioValue - (loanBalance / maxLTV)
- percentUntilMarginCall = 1 - (currentLTV / maxLTV)
- Negative values mean already in that zone

**calculateDropToMarginCall(state: SBLOCState, config: SBLOCConfig): number**
- Returns percentage drop in portfolio that would trigger margin call
- Formula: 1 - (loanBalance / (portfolioValue * maxLTV))

Create a new web component `AppHeader` extending `BaseComponent` with the following structure:

**Template:**
```html
<header class="app-header">
  <div class="header-brand">
    <div class="brand-wordmark">
      <span class="brand-icon" aria-hidden="true">âš¡</span>
      <h1 class="brand-title">eVelo</h1>
    </div>
    <p class="brand-tagline">Tax-Efficient Portfolio Strategy Simulator</p>
  </div>
  <div class="header-actions">
    <slot name="actions"></slot>
  </div>
</header>
```

**Styles (key points):**
1. Use CSS custom properties from tokens.css for all colors
2. Header background: `var(--color-primary)` for distinctive branding
3. Text color: `var(--text-inverse)` for contrast against teal
4. Fluid typography using `clamp()`:
   - Brand title: `clamp(1.25rem, 2.5vw + 0.5rem, 1.75rem)`
   - Tagline: `clamp(0.75rem, 1.5vw + 0.25rem, 0.875rem)`
5. Tagline: opacity 0.9 for subtle hierarchy
6. Responsive: Hide tagline at `@media (max-width: 768px)`
7. Header padding: `var(--spacing-md) var(--spacing-lg)`
8. Mobile padding: `var(--spacing-sm) var(--spacing-md)`
9. Action buttons container: flex with gap `var(--spacing-sm)`
10. Dark theme: Use `:host-context([data-theme="dark"])` selector for theme-aware adjustments

**Accessibility:**
- `aria-hidden="true"` on decorative icon
- Semantic h1 for brand title
- Proper color contrast (white on teal passes WCAG AA)

**Register:** `customElements.define('app-header', AppHeader)`

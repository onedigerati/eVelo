Add the following new CSS custom properties to the `:root` section of tokens.css:

**Widescreen Breakpoint Documentation (comments only - breakpoints used inline in components):**
```css
/* Widescreen Breakpoints (for reference, used in media queries):
 * --breakpoint-lg: 1440px (large desktop, 3-column viable)
 * --breakpoint-xl: 1920px (Full HD widescreen)
 * --breakpoint-2xl: 2560px (Ultrawide/4K)
 */
```

**Fluid Spacing Tokens:**
```css
/* Fluid spacing - scales smoothly between viewport sizes */
--spacing-fluid-sm: clamp(8px, 1vw, 12px);
--spacing-fluid-md: clamp(16px, 2vw, 24px);
--spacing-fluid-lg: clamp(24px, 3vw, 40px);
--spacing-fluid-xl: clamp(32px, 4vw, 56px);
```

**Fluid Typography Tokens:**
```css
/* Fluid font sizes - scales smoothly between viewport sizes */
--font-size-fluid-base: clamp(1rem, 0.95rem + 0.25vw, 1.125rem);
--font-size-fluid-lg: clamp(1.125rem, 1rem + 0.5vw, 1.375rem);
--font-size-fluid-xl: clamp(1.25rem, 1rem + 1vw, 2rem);
--font-size-fluid-2xl: clamp(1.5rem, 1.25rem + 1.25vw, 2.5rem);
--font-size-fluid-hero: clamp(2rem, 1.5rem + 2vw, 3.5rem);
```

**Widescreen Content Constraints:**
```css
/* Max content widths for widescreen readability */
--content-max-width-lg: 1400px;
--content-max-width-xl: 1800px;
--content-max-width-2xl: 2200px;
--chart-max-width: 1200px;
```

Place these tokens:
1. Add the breakpoint comments after the existing Layout section
2. Add fluid spacing after existing spacing tokens
3. Add fluid typography after existing typography tokens
4. Add content constraints in a new "Widescreen" section before Shadows

Also add these tokens to the dark theme `[data-theme="dark"]` section (same values - they don't change with theme):
- Copy all fluid spacing tokens
- Copy all fluid typography tokens
- Copy all content constraint tokens

WHY these specific values:
- clamp() provides smooth scaling between min and max based on viewport
- 1vw = 19.2px at 1920px, so clamp(16px, 2vw, 24px) gives 16px at 800px, 24px at 1200px+
- Content max-widths prevent excessive stretching on ultrawide (research recommends 1200-1400px for charts)
- Font fluid-hero matches research recommendation for hero metric values

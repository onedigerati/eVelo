Recommendation: Option B
Symbol Primary is the most elegant solution because:
  • Ticker symbols (AAPL, AVGO, BRK.B) are always 1-5 characters — predictable width
  • Industry standard — traders and investors recognize tickers instantly
  • Full name is still visible as secondary info
  • Works on all screen sizes without hover interactions
  • Clean visual hierarchy: Symbol → Name → Stats
